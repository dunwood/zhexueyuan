# 价值观的根：从经验、大脑到基因

从经验、大脑到基因

——《价值观是你活过的人生》续篇

续篇引言：追问还没有结束

上一篇文章我们得出了一个结论：价值观是一个人活过的人生的沉淀，它不是观点，而是身份本身。正因如此，它只能被共鸣，无法被辩论。

但这个结论本身，还留着一个更深的问题没有回答：

价值观究竟从哪里来？它的根，到底有多深？

如果它只是生活经验的积累，那理论上，只要改变一个人的生活环境，就能改变他的价值观。很多相信教育和启蒙的人，正是持有这种信念——他们认为，只要给对方足够的信息和理性引导，价值观终究是可以被改变的。

但现实一次次地证伪了这个信念。你给他看了所有的证据，他还是那个他。你和他争论了多年，他不但没有被说服，反而越来越坚定。

这不是因为他顽固，不是因为他愚蠢。而是因为价值观的根，比我们以为的深得多。

这篇续篇，将沿着三个层次递进地追问：

第一，社会学与经济学怎么看待信念与行为的关系？

第二，神经科学告诉我们，大脑结构与价值倾向之间有什么联系？

第三，也是最深、也最需要谨慎对待的一个问题——基因，是否参与了塑造价值观？

这是一趟越走越深、越走越让人感到震动的思想之旅。

第一层：社会学与经济学的发现——信念与行为，同根而生

1.1 帕累托的洞察：那棵树的比喻

二十世纪初，意大利社会学家维尔弗雷多·帕累托（Vilfredo Pareto）在其巨著《普通社会学纲要》中，提出了一个至今仍被低估的洞见。

他分析了人类行为的四种状态：

心理状态（A）、

崇拜行为（B）、

感情表现为信念（C），以及

外显行为（D）。

在分析这四者的关系时，帕累托指出了一个人们最容易犯的错误：我们习惯于认为，人们这样行动，是因为他们这样信仰。也就是说，我们默认信念决定行为，C决定D。

帕累托说，这个命题几乎总是错误的。

反过来说，人们这样信仰，是因为他们这样行动，似乎包含了更多的真理，但也不完全对。

帕累托给出了第三个也是最准确的命题：人们做出行动D并信仰C，两者并非因果，而是共同植根于同一个心理状态A。

信仰和行为并不一定就是独立的，如果它们之间有某种联系，那么这种联系则依赖于其内在性的关联，就象同一棵树上的两个分枝，仅仅是在同一棵树上而已。——帕累托

这个比喻，是整篇续篇的起点。那棵树，就是一个人深层的心理状态——由生命经历、情感记忆、文化浸润共同构成的内在结构。信念（价值观）和行为，都只是这棵树长出来的枝，你试图通过改变枝来改变树，是徒劳的。你必须去动那个根。

而根，是不轻易被触动的。

1.2 戴维森与修珀斯：信念与偏好，在科学上无法分离

帕累托的洞察是定性的。半个世纪后，美国哲学家唐纳德·戴维森（Donald Davidson）与斯坦福大学的帕特里克·修珀斯（Patrick Suppes）用实证研究，为这个洞察提供了科学支撑。

戴维森后来成为美国心灵哲学和语言哲学的核心人物，而修珀斯则是一位罕见的跨界天才，研究领域跨越主观概率论、决策行为学、量子物理、数理心理学和语言脑科学，被誉为斯坦福大学文艺复兴式的学者，并与诺贝尔经济学奖得主肯尼斯·阿罗（Kenneth Arrow）是超过六十年的挚友。

1955至1956年间，戴维森与修珀斯联名发表了两篇重要论文，核心议题是：一个人的信念与他的偏好（行为倾向），是否可以在实证上被分离？

他们的结论是：不能。

这个结论看似简单，实则深远。它意味着：你无法在不改变一个人的整体信念系统的前提下，单独改变他对某件事的偏好；同样，你也无法在不改变他的偏好结构的前提下，单独纠正他的某个信念。

用更直白的话说：一个人的价值观不是一块可以单独替换的零件，它是一个有机整体，改动任何一部分都会牵动整个系统。这就是为什么用单点的论据去说服一个人的价值观，几乎从来不奏效——因为你撬动的那个点，被整个系统牢牢锁住了。

更重要的是，这套信念系统，是在生活中形成的，而不是在头脑里推理出来的。经济学家兼思想家汪丁丁在解读这一研究时指出：我们其实只能在生活中形成我们的信念。信念，很可能就是既有的生活习惯。因为只有基于经验，信念才可能有所表达。稳定的经验，就转化为生活习惯，而基于习惯的行为，已经不再需要理性计算，它们是自发的。

你以为你的价值观是你思考出来的，实际上它是你活出来的。思考只是给它穿上了一件逻辑的外衣。

1.3 行为经济学的补充：禀赋效应与损失厌恶

2017年诺贝尔经济学奖得主理查德·塞勒（Richard Thaler）用毕生研究揭示了一个与此高度相关的行为规律：禀赋效应（Endowment Effect）。

禀赋效应的核心发现是：人们对自己已经拥有的东西的估价，远高于他们为获得同样东西愿意支付的价格。一旦某样东西成为你的一部分，失去它所带来的痛苦，会远大于得到它所带来的快乐。

这个规律，在经济行为上成立，在价值观上同样成立，甚至更加剧烈。

一个人的价值观，是他在几十年的生命中一点一点积累下来的。它是他最深层的禀赋。当有人试图说服他放弃或修正这个价值观，在心理层面上，他感受到的是一种极其真实的失去——失去的不只是一个观点，而是与这个观点紧紧相连的自我认同、生命记忆、以及由此做出的所有选择的意义。

损失厌恶（Loss Aversion）告诉我们，人对损失的敏感程度，大约是对同等收益敏感程度的两倍。在价值观层面，这个比例只会更高。这就是为什么，即使你的论据在逻辑上无懈可击，对方仍然会用尽全力维护他的立场——因为放弃，代价太大了。

第二层：神经科学的发现——价值观写在大脑里

2.1 大脑结构与政治倾向的关联

如果说社会学和经济学的发现还停留在行为层面，那么神经科学的研究则更进一步，直接打开了大脑，去看价值倾向在生理层面的基础。

越来越多的神经影像学研究发现，一个人的脑结构——不是思想，是物理意义上的大脑结构——与他的政治和道德倾向之间，存在统计上显著的关联。

其中最具代表性的一项研究发现：倾向于保守主义的人，大脑中与威胁感知和恐惧反应相关的杏仁核（amygdala）体积更大；而倾向于自由主义的人，与复杂性处理和不确定性接受相关的前扣带皮层（anterior cingulate cortex）灰质更多。

这不是政治观点的判断，也不是说哪种大脑更好。这是一个中性的生理事实：不同的大脑结构，导致对同样的世界信号产生不同的感受阈值和反应模式——对威胁更敏感，还是对新奇更开放？对秩序更倚重，还是对变化更包容？

这些感受方式的差异，是价值观形成最早的土壤。

你对世界的感受方式，在你开口说第一个字之前，就已经被你的大脑结构部分地决定了。

2.2 海特的道德模块理论：六个内置的道德感受器

纽约大学社会心理学家乔纳森·海特（Jonathan Haidt）在《正义之心》（The Righteous Mind）中，提出了一个影响深远的理论：人类大脑中内置了六个基本的道德模块，它们像感受器一样，对生活中出现的各种事件进行模式识别，并自动触发道德判断。

这六个模块是：

关爱与伤害、

公平与作弊、

忠诚与背叛、

权威与服从、

圣洁与堕落、

自由与压迫。

海特的关键发现是：每个人都拥有这六个模块，但每个人对这六个模块的权重排序不同。有人的关爱模块极度发达，有人对忠诚和权威格外敏感，有人对自由的渴望压倒一切。

这种权重差异，造成了人与人之间在道德直觉上的根本分歧。当不同权重组合的两个人讨论同一个道德问题，他们的感受出发点就不同，因此他们的结论几乎注定不同。

更重要的是，海特通过大量实验证明：道德判断是直觉在先，理性辩护在后。人们先有了感性的直觉反应，然后才调用理性去寻找支持这个反应的理由。理性，在这里扮演的不是法官的角色，而是辩护律师的角色。

他用一个让人印象深刻的实验说明了这一点：实验者先通过催眠，让受试者在看到某个特定词语时产生恶心感。醒来后，受试者在判断含有该词语的道德题时，会因为不明原因的恶心感而认定那件事不道德——然后他们会认真地为这个判断找理由，尽管那个理由根本与词语无关。

你以为你在用理性判断是非，实际上你的大脑早已经作出了判断，理性只是在事后帮你写答辩词。

这意味着，当你向一个人呈现逻辑严密的论据，试图改变他的道德判断，你面对的不是一个等待被说服的理性主体，而是一个已经作出判断的直觉系统——而且这个系统有一整套理性辩护机制来保护自己的判断。

2.3 神经经济学：当价值观进入大脑扫描

神经经济学（Neuroeconomics）是一个新兴的交叉学科，它把经济学的决策模型和神经科学的脑成像技术结合起来，直接观察人在做价值判断和选择时，大脑发生了什么。

这一领域的研究积累了一个重要发现：人在做道德相关的决策时，激活的脑区与做纯粹逻辑运算时激活的脑区，存在显著差异。道德判断强烈激活的是与情感处理相关的脑区，包括前额叶皮层、杏仁核、岛叶皮质等，而非单纯的逻辑推理区域。

换句话说，价值观问题在大脑里的处理路径，天生就不是逻辑路径，而是情感路径。你用逻辑去攻打一个情感堡垒，武器本身就用错了。

还有一个更值得注意的发现：当一个人的核心价值判断受到挑战时，大脑激活的模式与身体受到威胁时的激活模式高度相似。价值观的冲突，在神经层面，就是一种真实的威胁感。这不是比喻，这是生理事实。

这解释了为什么价值观讨论如此容易升温、如此难以理性维持——因为对话双方的大脑都已经进入了防御模式，而防御模式下的大脑，对外来信息的开放程度会大幅下降。

第三层：基因的问题——最深的根，也是最危险的话题

3.1 为什么这个问题危险，但不能回避

现在我们来到这篇文章最深、也最需要谨慎的地方：基因与价值观的关系。

这个话题之所以危险，是因为它在历史上曾被严重滥用。二十世纪上半叶，社会达尔文主义和优生学打着科学的旗号，用基因决定论为种族歧视和政治压迫提供理论依据，造成了人类历史上的重大悲剧。这段历史，使得任何涉及基因与人类行为关系的讨论，都需要极度谨慎。

但回避，并不等于严肃。

科学在继续前进。过去三十年里，行为遗传学（Behavioral Genetics）积累了大量证据，表明基因对人类行为、气质和倾向的影响，是真实存在且不可忽视的。否认这一点，不是道德立场，而是反科学。

关键是：我们如何理解这个影响的性质？它是决定论式的，还是概率性的？它是直接的，还是通过中间机制发生作用的？

3.2 基因不决定价值观，但塑造了感受价值观的方式

海特在《正义之心》中明确指出：政治意识形态有其先天基础，这一点已经得到了来自行为遗传学的多项研究支持。

双胞胎研究是这一领域最有力的方法之一。通过对比同卵双胞胎（基因几乎完全相同）和异卵双胞胎（基因相似程度与普通兄弟姐妹相当）在政治倾向上的相似性，研究者可以估算遗传因素的贡献比例。

多项研究的结果高度一致：政治倾向的遗传率（heritability）约在40%至60%之间。这意味着，在解释人与人之间政治倾向差异的诸多因素中，基因的贡献大约占到一半。

但这里有一个极其重要的区分，必须清楚地说明：

基因并不直接编码某种具体的政治立场或价值观。没有一个基因负责让你成为自由主义者，也没有一个基因负责让你成为保守主义者。

基因的作用，是塑造了你的气质倾向——你对威胁的敏感程度、你对新奇体验的开放程度、你对不确定性的容忍能力、你的同理心强度、你对权威的本能态度。

这些气质倾向，决定了你在成长过程中如何感受同一个世界——你注意到什么，忽略什么；什么让你感到安全，什么让你感到威胁；什么让你感到兴奋，什么让你感到恶心。

这些感受方式，经年累月地积累，就塑造了你的价值观。

基因不是价值观的直接作者，但它是决定你用什么样的眼睛去看世界的那个设计师。用不同眼睛看同一个世界，看到的是不同的世界，形成的是不同的价值观。

3.3 基因的表达：它寻找适合自己的环境

行为遗传学还有一个更精妙的发现，常常被通俗讨论所忽略：基因与环境不是彼此独立的两个变量，它们相互作用，而且基因会主动影响人所处的环境。

这叫做基因-环境相关性（Gene-Environment Correlation）。一个对新奇体验天生充满渴望的人，会主动寻找刺激性更强的环境；一个对威胁天生更敏感的人，会倾向于留在他熟悉、感到安全的社群中。他们各自选择的环境，又进一步强化了他们的气质倾向，从而深化了各自的价值观。

如文章开头提到的那段文字所说：特定的基因会让你主动去寻找适合这个基因发展的环境。一个自由主义倾向的人，会天然地对权威秩序感到不适，主动去亲近多元开放的文化；一个保守主义倾向的人，会天然地对传统和稳定感到安心，主动去强化这种环境的联结。

这不是屁股决定脑袋，这是脑袋决定了屁股，而脑袋的底层配置，有一部分在出生时就已经写好了。

更值得注意的是这样一个现象：当你刚认识一个人，发现他的某个观点你非常认同，随着了解加深，你会发现你们之间的共同点越来越多。这种感觉，很可能不是巧合，而是基层气质结构高度相似的两个人，在各自不同的人生路径上，走向了高度相似的价值观。

3.4 说服的边界：基因视角下的重新理解

理解了基因的这层作用之后，你说服他人改变价值观的企图，就有了一个更深层的解释为什么几乎注定无效。

当两个人的气质底层差异很大，他们对同一个世界的基础感受方式就不同。你的论据，是基于你的感受方式建立的；他的抵触，是基于他的感受方式产生的。你们不只是观点不同，你们感知世界的方式就不同。

这不是任何一方的错。这是两种不同的人，在同一个世界上共存的事实。

当然，这里必须再次强调：基因是概率性的，不是决定论式的。它提供的是倾向，不是命运。人生经历可以大幅修正基因提供的初始倾向（表观遗传学），重大事件、深度关系、跨文化的生活经验，都可能让一个人的价值观发生真实的转变。

但这种转变，往往需要的不是一场说服，而是一段生活。不是一个论据，而是一种沉浸式的经历改变了他感受世界的方式，从而在根本上松动了他的价值观基础。

你无法用语言改变一个人的价值观，但生活可以。那需要的不是一场对话，而是岁月。

第四层：整合——价值观形成的完整图景

4.1 四个层次的嵌套

现在，我们可以把价值观的形成图景，完整地描绘出来。它不是单一的，而是四个层次相互嵌套的结果。

最深的一层，是基因赋予的气质底色——对威胁的敏感度，对新奇的渴望度，对他人的共情能力，对权威的本能态度。这是你感知世界的初始设置。

第二层，是大脑结构与功能——在成长过程中，基因与早期环境共同塑造了大脑的物理结构，包括杏仁核的大小、前额叶皮层的厚度、神经连接的密度与模式。这些结构，决定了你的情感处理方式和道德直觉的触发阈值。

第三层，是生活经历的积累——在你的气质底色和大脑结构的基础上，你经历了具体的人生：童年的家庭环境，青少年时期的社会挫折或成功，成年后的重大事件和关键关系。这些经历，一层层地沉积，成为帕累托所说的那棵树的树干和主枝。

最表层，才是可见的价值观表达——你说出来的信念，你做出的选择，你对时政的判断，你对他人行为的道德评价。这是那棵树上的叶子，是可见的，但它由底下三层决定。

当两个人争论价值观，他们只是在争论对方树上的叶子。而那棵树的树根，根本没有被触动。

4.2 互联网时代的新变量

汪丁丁在他的文章中提到了一个值得认真对待的现象：互联网，虽然是虚拟社会，但它借助语言，仍然是我们关于真实世界信念系统的一种表达方式。互联网上的经验，也会融入并影响我们在真实社会生活中形成的信念。

但互联网还有另一个更深的影响，汪丁丁也点到了：在几百万年的演化过程中，人类习惯了不足百人的日常交往范围。现代人的大脑，尽管已经远超其他灵长类，仍然无法直觉地处理互联网时代动辄数亿人的社交规模。

这造成了一个吊诡的结果：连接越广泛，理解越浅薄。你在网上认识了来自各地、各种背景的人，但大脑处理这种规模连接的能力，远不足以产生真实的理解。于是，你接触到了海量的不同价值观，却只是更加确认了自己的价值观——因为算法把与你相似的声音推给你，把不同的声音过滤掉；同时，你的大脑对陌生人的不同价值观，缺乏换位理解所需的情感连接基础。

互联网时代的价值观冲突，比历史上任何时代都更频繁、更激烈、更难以弥合。不是因为人变得更顽固了，而是因为在一个每天接触数百万人的世界里，你的大脑根本没有为处理这种规模的价值多样性做好准备。

结语：深知之后的态度

不是虚无，是深刻的尊重

读到这里，也许你会有一种沮丧感：如果价值观的根如此之深，深到基因，深到大脑结构，那么改变是否根本不可能？沟通是否根本无意义？

这个结论是错的。

我们追问价值观的根有多深，不是为了得出一个宿命论的结论，而是为了得到一种更深刻的谦逊。

当你知道对方价值观的形成，不是他的任性，不是他的愚蠢，不是他被操纵了——而是他的基因、他的大脑、他的整段人生共同铸造的结果，你对他的判断会发生根本性的变化：从评判，到理解；从说服的冲动，到接纳差异的平静。

帕累托的那棵树的比喻，在这里有了它最深的意义：不同的树，长在不同的土地上，接受不同的阳光和雨水，长出不同的形状。没有哪棵树是错的，它们只是不同的树。

你能做的，不是砍掉另一棵树，让它长成你的样子。你能做的，是去寻找那些与你生长在相似土地上的树，那些在根的层面与你同频的人。与他们一起生长，这才是真正可能发生的共鸣。

理解了价值观的根有多深，你就不会再急于改变别人。你只会更珍惜那些无需改变就能彼此理解的人。

这不是冷漠，这是经过深思之后的尊重。

尊重他人价值观的深根，也尊重你自己的。
