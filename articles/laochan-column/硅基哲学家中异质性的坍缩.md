硅基样本正越来越多地被用作人类小组的低成本替代品，并且已经被证明能够以较高的保真度再现人类总体意见。我们表明，在与对齐相关的哲学领域中，硅基样本会系统性地坍缩异质性。我们使用来自 PhilPeople 个人资料的 (N = 277) 名专业哲学家的数据，评估了七种专有和开源大语言模型在两个方面的能力：一是复制个体哲学立场，二是在不同哲学领域之间保留跨问题的相关结构。我们发现，语言模型会显著过度关联哲学判断，从而在不同领域之间制造出人为的共识。这种坍缩部分与专家效应有关，即模型会隐含地假定某一领域的专家拥有高度相似的哲学观点。我们通过研究 DPO 微调的影响，并将结果与完整的 PhilPapers 2020 调查（(N = 1,785)）进行验证，来评估这些发现的稳健性。最后，我们讨论这些发现对于对齐、评估，以及使用硅基样本替代人类判断的影响。本项目的代码可见于 https://github.com/stanford-del/silicon-philosophers。

### 1 引言

硅基抽样——即根据人口统计信息和态度信息对大语言模型（LLMs）进行条件设定，以生成合成的人类回答——为社会科学研究中的人类小组提供了一种低成本替代方案。Argyle 等人 [2] 证明，GPT-3 在模拟美国政治意见时表现出算法保真度，它不仅能够复制个体偏好，还能够复制态度之间复杂的相关模式。Park 等人 [18] 通过基于访谈的条件设定扩展了这些发现，实现了与人类重测信度相当的准确率。

然而，近期研究开始质疑，硅基样本是否能够忠实保留人类群体内部的异质性。Santurkar 等人 [21] 发现，LLM 的意见会系统性地偏离美国普通人群。Durmus 等人 [7] 表明，模型很难忠实代表少数群体观点。Wang 等人 [29] 证明，LLM 会压平身份群体，放大刻板印象，而不是捕捉真实的多样性。这些发现表明，在本来存在真实分歧的地方，硅基抽样可能会制造出人为的共识。

专业哲学为检验硅基抽样是否能够在由专家主导、但充满分歧的共同体中保留异质性，提供了一个理想案例。PhilPapers Survey [5] 记录了 1,785 名哲学家在形而上学、认识论、伦理学、心灵哲学以及其他子领域中的立场。该调查揭示了结构化的分歧：哲学立场表现出强相关结构，例如物理主义者往往倾向于无神论和自然主义，而不同问题上的一致程度也存在很大差异。

本文研究的是，当 LLM 模拟哲学家时，是否能够保留这些模式，包括它们是否能够维持在人类群体中观察到的自然异质性，以及是否能够复制哲学立场之间的相关结构。我们发现，LLM 会系统性地坍缩异质性：它们会过度关联哲学判断，制造人为共识，并表现出虚假的专家效应。

在这种虚假的专家效应中，模型会假定某一领域的专家持有刻板化地一致的观点。我们考察了表征保真度的两个互补维度：

**RQ1（异质性与结构对齐）：**硅基样本是否表现出与人类哲学家相同幅度和结构的哲学分歧？

**RQ2（相关结构保留）：**硅基样本是否保留了哲学立场之间经验上观察到的相关结构？微调是否能够改善这种保留？

为回答 RQ1，我们考察总体方差水平、领域层面的可预测性、人口统计特征与立场之间的相关性，以及分歧的潜在维度，也就是主成分，在 LLM 与人类之间是否对应。为回答 RQ2，我们测试 LLM 是否能在不同问题之间维持内部一致性。人类哲学家在相关立场之间表现出显著相关性；如果模型能够正确预测个别答案，却不能保留这些相关性，那么这可能表明它们缺乏真正的哲学推理。

这些问题共同具有 AI 对齐方面的意义：如果 LLM 会坍缩专家哲学家的异质性，而这些哲学家可以说是最有能力就对齐议题提供判断的人，那么模型就有可能在本来存在真实分歧的地方强加人为共识。我们的发现显示，LLM 在这两个维度上都存在系统性失败：与人类哲学家相比，LLM 产生的方差低 1.9 到 3.9 倍，并且会沿着根本不同的轴线组织分歧。尽管使用直接偏好优化（DPO）进行微调能够改善相关结构的保留，但它并不能解决底层的异质性坍缩问题。

本文包括以下贡献： （1）在一个专家领域中对算法保真度进行系统性测试； （2）按照 Bourget 和 Chalmers [5] 的方法，提出一个使用主成分分析来评估异质性保留的新框架； （3）识别出所有被测试模型中都存在的系统性过度相关和虚假专家效应； （4）分析微调是否能够补救这些问题。

### 2 相关工作

**硅基抽样。**Argyle 等人 [2] 通过四项标准来操作化算法保真度：

（1）**社会科学图灵测试**——生成的回答与人类文本无法区分； （2）**向后连续性**——回答与社会人口统计条件背景保持一致； （3）**向前连续性**——回答从上下文中自然推进，反映出适当的形式和语气； （4）**模式对应**——回答反映出在人类数据中观察到的观念、人口统计特征与行为之间的底层关系模式。

模式对应与本文最为相关，因为它测试的是模型是否捕捉到结构化的信念系统，而不只是孤立的回答。他们关于美国选民模拟的研究表明，LLM 能够复制政治态度、人口统计特征与行为之间复杂的相关性。Park 等人 [18] 通过对 1,052 名个体进行基于访谈的条件设定，进一步扩展了这一研究，并实现了 85% 的准确率，这与人类重测信度相当。

除政治意见之外，Aher 等人 [1] 提出了“图灵实验”，证明模型能够再现最后通牒博弈和囚徒困境中已知的人类行为模式。Filippas 等人 [8] 提出了**homo silicus**，即基于 LLM 的模拟经济主体，并展示了它们在经济学试点实验中的用途。Sarstedt 等人 [22] 为消费者研究中的硅基抽样提供了指导原则，并指出结果在不同领域之间差异很大。然而，这些研究考察的是普通人群在日常主题上的表现，而这些主题是非专家也能够理解和回答的。

**人格提示。**尽管硅基抽样在总体模拟方面显示出前景，但关于群体内部异质性的担忧已经出现。Santurkar 等人 [21] 记录了 LLM 意见中系统性的自由主义倾向。Durmus 等人 [7] 表明，意见分布在不同模型之间差异很大，并且可以通过提示加以改变，但少数观点仍然代表不足。Wang 等人 [29] 提供了异质性坍缩的直接证据，发现 LLM 会减少群体内部差异，并放大刻板印象，而不是捕捉真实的多样性。Sourati 等人 [26] 认为，LLM 正在大规模同质化人类表达与思想，在实验室环境之外重塑语言规范。Bisbee 等人 [3] 在调查领域证实了这些担忧，表明 LLM 生成的回答会系统性地扭曲意见分布。Gao 等人 [10] 进一步记录到，LLM 会将实验操控中的效应量高估二到十倍，因此警告不要将其用作人类替代品。

近期研究已经开始探索通过微调来缓解这些缺陷。Suh 等人 [27] 使用来自 70,000 个人口统计画像的回答分布，对 3,362 个调查问题进行微调，从而改善分布对齐 [25]。Hewitt 等人 [11] 使用 LLM 预测社会科学实验结果。Kolluri 等人 [14] 使用 290 万条社会科学回答，对 Llama-3.1-8B 和 Qwen-2.5-14B 同时进行监督微调（SFT）和直接偏好优化（DPO）[19]，并发现 DPO 在个体层面预测准确率上表现最好，且能够稳健泛化到未见过的研究。然而，Kirk 等人 [13] 表明，与 SFT 相比，基于人类反馈的强化学习（RLHF）会显著降低输出多样性，这意味着对齐与多样性之间存在一种根本性的权衡。我们的工作不同之处在于，我们关注的是一个专家领域，也就是哲学；在这个领域中，需要研究的现象不是共识，而是结构化分歧。

**哲学分歧。**PhilPapers Surveys [4, 5] 记录了 1,785 名专业哲学家在 100 个问题上的立场，为研究专家立场提供了异常丰富的数据。具体而言，该调查揭示了哲学观点之间的相关性、专家效应，以及一种并不沿单一意识形态轴线组织起来的结构化分歧，这一点是通过主成分分析（PCA）识别出来的。Schwitzgebel 和 Cushman [23] 记录到，专业哲学家尽管拥有专业知识，仍然表现出认知偏差，这表明哲学推理结合了系统性框架与持续存在的启发式判断。Cheung 等人 [6] 证明，LLM 在道德决策中表现出被放大的认知偏差——其中遗漏偏差比人类更强——这表明带有规范负荷的领域可能尤其容易受到扭曲。虽然这些研究考察了 LLM 在哲学推理中的偏差，但没有任何研究系统性地检验硅基抽样是否保留了哲学家之间显现出来的结构化分歧。

### 3.2 评估

我们采用来自生态学、信息论和多变量统计学的互补指标，每一种指标捕捉保真度的不同方面。

**专家效应。**我们使用两个来源来验证 LLM 的专家效应：PhilPapers 2020 Survey（(N = 1,785)）以及我们自己的 277 名哲学家的真实基准数据。双来源验证为虚假关联提供了更强证据，避免将样本特定的人为结果与真正的 LLM 偏差混淆。对于每一种专家效应，我们使用带有 Yates 校正的卡方统计量。如果真实基准数据没有显著性（(p > 0.05)），但 LLM 显示出显著关联，那么该效应就是虚假的。

**主成分分析。**为了刻画哲学分歧的维度和结构，我们按照 Bourget 和 Chalmers [5] 的方法应用主成分分析：对于每一个问题，我们选择一个数值变量；对于非二元问题，选择人类回答者中最受欢迎的选项。由此，每个数据来源得到 100 个变量。

PCA 是合适的，因为哲学立场并不是彼此独立的。信念会形成具有系统性相关模式的连贯世界观，而 PCA 可以揭示 LLM 是保留了这种结构化分歧，还是将其坍缩为人为共识。Bourget 和 Chalmers [5] 使用 R 的 missMDA 包进行插补；我们沿用其思路，应用迭代 PCA 插补：首先使用列均值初始化缺失值，然后使用 5 个成分拟合 PCA，重构并只替换缺失条目，之后反复迭代直到收敛。这避免了整列删除造成的信息损失，同时生成了一个适用于标准 PCA 的完整数据矩阵。此外，数据插补只用于 PCA；问题相关结构分析，也就是第 4.4 节，使用成对删除的相关矩阵，以避免引入人为结构。

人类回答具有大量缺失值，在补集恢复之后缺失率为 61.1%；而 LLM 数据的缺失率较低，根据模型不同为 17%–40%。迭代 PCA 插补通过利用每个数据集内部的相关结构来填补缺失值，而不是编造彼此独立的立场，因此可以处理这种不对称。

按照 Bourget 和 Chalmers [5] 的方法，我们统计每个至少解释 2% 方差的成分。虽然从数学上说，PCA 会提取与变量数量一样多的成分，但只有超过这一阈值的成分，才代表有意义的分歧维度，而不是噪音。我们报告以下内容：

（1）每个成分解释的方差； （2）超过 2% 阈值的成分数量； （3）每个成分中载荷最高的问题； （4）在最优对齐之后，人类成分与 LLM 成分之间的载荷相关性。

### 3.4 微调

为了研究微调是否能够改善结构保真度，我们对 Llama-3.1-8B 应用 DPO [19]。DPO 特别适合这一领域，因为哲学立场是有根据的偏好问题，而不是客观真理问题——哲学家认可某些立场，是基于经过推理的判断，而不是基于事实正确性。不同于 SFT 将哲学立场视为真实标签，DPO 的偏好学习框架更好地捕捉了哲学分歧的性质。我们从经验上验证了这一选择：在同一数据上进行 SFT 会产生严重的模式坍缩，84% 的回答默认变为“接受”；见附录 C.6.2。

我们观察到两种模式，并使用针对 277 名哲学家真实基准数据的 (t)-检验验证每一种模式。²

**匹配相关性。**在我们的真实基准数据中，宗教哲学专家支持有神论宇宙论论证的比例，相比非专家高出 40.3 个百分点（(p < 0.01)），所有 LLM 都复制了这一方向，例如 Claude-Sonnet-4.5 为 60.1 个百分点，(p < 0.001)，准确反映了强烈的人类模式。

**被放大的相关性与虚假相关性。**LLM 会系统性地将人类中较弱或不存在的相关性夸大为强而显著的相关性。形而上学专家对于休谟式自然法则表现出一种较弱且不显著的倾向（−8.2 个百分点，(n = 54)），但 LLM 平均将其放大到 −19.3 个百分点，七个模型中有四个在 (p < 0.001) 水平上显著。在更极端的情况下，某些相关性在 LLM 预测中显著出现，但在人类数据中完全不存在。表 3 按照第 3.2 节描述的双来源验证方法，呈现了候选的虚假效应。

“生物哲学 → 生物学式个人同一性”的相关性说明了这一点：我们的真实基准数据只显示一种较弱且不显著的关联（11.4 个百分点，(n = 12)），PhilPapers 2020 Survey 也报告这一配对不存在显著专家效应，然而七个 LLM 中有四个预测出很大的正向差异，平均约为 43 个百分点，(p < 0.001)。这种虚假预测可能是由“biology”和“biological”之间的表层词汇重叠造成的；这种模式在不同架构之间都保持一致。类似地，LLM 预测古代哲学专家会强烈支持亚里士多德式实践理性，平均差异为 68.9 个百分点，所有 7 个模型均显著；而我们的真实基准数据仅显示 1.9 个百分点，且不显著。

这些例子表明，LLM 会锚定在人口统计标签上，并据此分配刻板化立场，未能恢复在人类专家中可见的不可知态度与跨领域复杂性。最能由人口统计特征预测的问题列于附录 C.1。

这些发现与 Santurkar 等人 [21] 的研究一致，后者表明 LLM 意见会系统性地偏离人类群体；也与 Durmus 等人 [7] 的研究一致，后者记录了全球意见表征中的偏差。我们的结果将这些担忧扩展到了专家领域。

### 4.3 主成分分析：结构对齐

按照 Bourget 和 Chalmers [5] 的方法，我们应用 PCA 来刻画哲学分歧的维度，并评估人类回答与 LLM 回答之间的结构对齐（RQ1 继续）。表 6 总结了所有来源的 PCA 结果。

**人类 PCA 结构。**人类哲学家表现出 5 个主成分，每个主成分解释至少 2% 的方差；前 6 个成分解释总方差的 71.1%。PC1（20.2%）在元哲学上载荷最强：自然主义（+0.31）、道德：自然主义（+0.28）、元伦理学：道德实在论（−0.28）。PC2（16.0%）捕捉延展心灵（+0.41）、环境伦理学（−0.33）以及心灵内容：外在主义（+0.31）。PC3（14.0%）在不朽（+0.42）、法律：法律实证主义（+0.32）以及维特根斯坦：后期（−0.27）上具有较高载荷。这种多维性表明，分歧沿着多个独立轴线发生，没有任何单一维度占据主导地位；这与 Bourget 和 Chalmers [5] 的发现一致，即哲学观点并不是沿单一的“左—右”谱系组织起来的。完整的成分解释见附录 C.3。

**LLM 与人类第一主成分（PC1）的比较。**人类 PC1 跨越元伦理学、元哲学、道德以及宗教哲学，而 LLM 的 PC1 显示出更强的主题聚类：GPT-4o 在心灵上传和时空问题上载荷很重，而 Llama-3.1-8B 则聚集了外部世界怀疑论和指称理论。Claude-Sonnet-4.5 与人类载荷的 PC1 对齐最好（(|r| = 0.63)），其最高载荷中共享了元哲学与道德：自然主义。各模型的 PC1 载荷见附录 C.3.2。

**结构对齐指标。**我们使用三种互补指标来评估结构对齐：逐元素相关性，即问题配对是否以相似方式相关；载荷相关性，即问题是否具有相似的 PCA 载荷；以及问题重叠，即最高载荷问题是否匹配。完整指标定义见附录 C.3。

大多数模型的前 6 个成分捕捉到的方差高于人类，例如 GPT-4o 为 83.0%、GPT-5.1 为 81.6%、Mistral-7B 为 73.8%，而人类为 71.1%。这表明 LLM 的分歧坍缩到了更少的维度上，与异质性坍缩一致。开源模型 Llama-3.1-8B（62.4%）和 Llama-3.1-8B (FT)（61.2%）捕捉到的方差低于人类，这表明这些模型将方差更分散地分布在多个成分中。逐元素相关性从弱到中等不等（0.016–0.245），其中 Claude-Sonnet-4.5 的结构对齐最好（0.245），其次是 GPT-4o（0.103）和 GPT-5.1（0.099）。载荷相关性仍然很弱（−0.137 到 0.110）。人类 PC1 跨越元伦理学、元哲学和道德，而 LLM 的 PC1 则按表层主题聚类，例如 GPT-4o 集中在心灵上传/时空问题上。附录 C.3 中的逐成分比较显示，Claude-Sonnet-4.5 的 PC1 个体对齐最好（(|r| = 0.63)），多数成分低于 0.20。

### 4.5 微调的影响

表 8 比较了基础版与微调版 Llama-3.1-8B 在回答多样性和结构一致性指标上的表现。微调模型更好地保留了问题之间的相关性，但也使总体回答分布进一步偏离人类。相关性 KL 从 0.195 改善到 0.176，下降 10%；而相关性 JS 保持不变，仍为 0.046。逐元素相关性翻倍，从 0.020 增加到 0.044，并在 (p < 0.05) 水平达到显著。平均熵增加，从 0.737 增加到 0.794，尽管回答完全一致的问题数量仍然相近，分别为 9 个和 10 个。然而，总体回答分布发生了明显偏移：微调模型将 34.7% 的回答分配给“不可知”，相比之下，基础模型为 1.5%，人类为 1.4%；回答 KL 从 0.07 增加到 0.40。

有三种机制可以解释这种权衡。第一，DPO 训练可能过拟合了训练分布中的不可知倾向。第二，有限的训练数据，即 3,434 个样本，可能不足以同时捕捉结构关系和题内差异。第三，对比式 DPO 目标可能倾向于削弱回答强度，而不是保留回答分布。我们将这一问题留给未来研究。逐题影响详见附录 C.6.1。

### 5 讨论

这些发现引发了对使用硅基样本作为专家判断代理的担忧。LLM 组织哲学分歧的维度与人类根本不同，它们会根据表层主题相似性来聚类问题，而不是根据人类数据中观察到的跨领域模式来组织问题。这表明，模型依赖的是词汇联想，而不是连贯的哲学推理；这一模式在所有七个模型中都保持一致，说明它反映的是 LLM 的系统性属性，而不是某个特定模型的人为结果。

被放大的相关性和虚假专家效应表明，模型应用的是刻板印象，而不是学习到的模式。我们的发现与 Wang 等人 [29] 的观察一致，即 LLM 会压平身份群体；在这里，这一现象被扩展到了专业哲学家身上。对于规范对齐至关重要的领域，扭曲最为严重：应用伦理学（RMSE 0.406，H.Var 0.100）和宗教哲学（RMSE 0.412，H.Var 0.097）在 14 个领域的可预测性排名中分别位列第 13 和第 14，见表 5；相对于人类分歧，LLM 方差大约坍缩了五倍。

刻板化与可预测性模式彼此强化。LLM 依赖人口统计标签来生成回答，见表 3，但这种机制只有在人类哲学家本来就已经趋于一致的地方才会成功。在那些专家分歧最有信息量的争议性问题上，同样的机制会产生不反映真实基准的虚假关联。DPO 微调部分缓解了结构错配，逐元素 (r) 翻倍，但并没有解决这种对共识的依赖，而且还引入了自身的分布扭曲，见第 4.4 节。

问题层面的可预测性主要由人类共识解释，而不是由模型能力解释：RMSE 与真实基准方差高度相关（(r = 0.75)）。在哲学家本来就已经达成一致的问题上，LLM 显得准确；但这掩盖了它们无法再现结构化分歧的能力缺陷，而这种结构化分歧恰恰定义了最具争议的哲学立场，并且可以说对 AI 对齐最为重要。

这些结果表明，尽管硅基抽样对于模拟总体人群趋势可能有用 [2]，但对于需要忠实表征专家分歧的应用来说，它可能是不充分的。研究者在使用 LLM 模拟规范负荷领域中的专家判断时，应当特别谨慎，并且应明确测量结构对齐，而不仅仅测量总体准确率。

**《硅基哲学家中异质性的坍缩》 • 13**

本文存在两方面局限。第一，我们的 277 名哲学家样本存在地理偏差，过度代表了拥有北美博士学位的哲学家；在 214 名博士授予国家已知的回答者中，这一比例为 73.8%；另有 22.7% 完全缺乏博士授予国家信息。虽然主题多样性，也就是 AOS 类别，得到了保留，但这些地理偏差可能影响结果的可推广性，尤其是对于那些不同地区哲学传统存在实质差异的问题而言。其他选择偏差也可能存在，因为维护活跃 PhilPeople 个人资料的哲学家，可能在系统上不同于其他哲学家，例如他们可能更常参与线上活动、更年轻、更具技术素养。自我报告的立场也可能自数据收集以来发生变化，从而引入时间上的不一致。

第二个局限来自我们数据的高缺失率，以及因此必须进行数值插补。我们的主成分分析使用迭代 PCA 插补，遵循 Bourget 和 Chalmers [5] 的方法，这相当于 R 的 missMDA 包。插补只用于 PCA；问题相关结构分析使用成对删除的相关矩阵，以避免引入人为结构。虽然插补利用相关结构来填补缺失值，但它必然引入基于模型的假设。LLM 数据的缺失率低于人类数据，分别为 17%–40% 和 61.1%，这造成了一种不对称：插补对人类数据的影响更大。我们的主要发现，也就是异质性坍缩和失败模式，对于这种不对称是稳健的，因为它们由多个不依赖插补的独立指标所确认，包括回答多样性、人口统计相关性和矩阵相似性。

### 6 结论

我们使用 277 名哲学家以及 PhilPapers 2020 Survey 对七个 LLM 进行评估，揭示了三个主要发现。

第一，所有 LLM 在哲学分歧的组织方式上都表现出结构错配。LLM 根据表层主题相似性来聚类问题，而不是根据人类数据中观察到的跨领域模式来聚类问题。专有模型用更少的主成分捕捉到更多方差，这与异质性坍缩一致。

第二，LLM 表现出被夸大的“人口统计—立场”相关性，并伴随虚假专家效应。这种“刻板化”行为——也就是模型锚定在人口统计标签上，以分配刻板化立场，而不是恢复专家的不可知态度——解释了它们为什么无法捕捉专家数据中的结构。

第三，DPO 微调改善了相关结构的保留，但使总体回答分布进一步偏离人类。这表明，在结构保真度与分布准确性之间，偏好优化面临内在权衡。

这些结果表明，对于需要在规范负荷领域中忠实表征专家分歧的应用来说，硅基抽样是不充分的。对齐研究必须考虑这样一个事实：LLM 可能会产生并不反映专家理性分歧的人为共识。

未来研究有几个方向。第一，使用官方 PhilPapers 个体数据进行验证，将加强关于真实基准模式的主张。第二，刻画温度—保真度权衡，将有助于澄清抽样多样性是否能够缓解异质性坍缩。第三，探索替代性微调方法，例如多样性正则化目标、改变 DPO 的 (\beta)，可以判断多样性—结构权衡究竟是内在的，还是可以被克服。最后，开发能够在准确性之外明确量化异质性保留程度的指标，将有助于改进硅基抽样的评估。
